# Base16 Dracula preview

A short **Markdown** sample for the theme.

## Features

- Headings in green
- **Bold** in pink
- _Italic_ emphasis
- [Link to repo](https://github.com/Francisco-BT/base16-dracula)

Inline `code` and a fenced block:

```json
{ "theme": "Base16 Dracula" }
```

> Blockquote — should read in aqua italic.

Regular paragraph with `inline code` and **bold** plus _italic_.
