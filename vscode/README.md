# Base16 Dracula Theme

Dark theme for VS Code: Base16 palette with a Dracula accent, aligned with the [Neovim colorscheme](https://github.com/Francisco-BT/base16-dracula) in the same repo (`colors/base16-dracula.lua`). Background tuned to **Rosé Pine Moon–style** `#232136`.

## Usage

1. **Command Palette** (`Ctrl/Cmd+Shift+P`) → **Preferences: Color Theme**
2. Pick **Base16 Dracula**

Semantic highlighting is supported where the language extension provides LSP tokens.
